# What's this?
This is a CLI tool for one line diary. You can record your daily with simple command.

# How to install
```sh
$ go install github.com/midry3/diajog/cmd/dj@latest
```

# Usage
```sh
$ dj <option> "Cotent"
```

# Options
### -h, --help
Show help message.

### -v, --view
View your diary.
When it's long, please use with `less` command.
